# README - Repositório de Aula

Este repositório foi criado para apoiar estudos práticos de **Git e GitHub** em aulas da **Rocketseat**, plataforma brasileira de educação em tecnologia reconhecida por conteúdos focados em desenvolvimento de software e aprendizado prático.

## Sobre a Rocketseat

A Rocketseat oferece trilhas e conteúdos para formação de desenvolvedores, com foco em prática, organização de projetos e uso de ferramentas do dia a dia do mercado, como controle de versão com Git e colaboração com GitHub.

## Conteúdos deste repositório (`conteudo.md`)

De acordo com o arquivo `conteudo.md`, este projeto:

- foi criado especificamente para aulas da Rocketseat sobre Git e GitHub;
- não possui funcionalidades avançadas nem uso de frameworks;
- contém apenas código JavaScript simples, sem uma aplicação real implementada.

## Objetivo

Servir como base de exemplo para praticar comandos, versionamento e fluxo de trabalho com Git/GitHub.
